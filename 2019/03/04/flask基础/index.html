Flask基础

框架,Flask

一、flask环境配置

1.使用pycharm创建
2.使用virtualenv创建（以下操作均在windows命令提示符下执行）
  • 第一步：安装

pip install virtualenv

  • 第二步：创建虚拟环境

virtualenv --no-site-packages flaskenv1(此处为环境名)

  • 第三步：激活环境

 a.windows下激活：

 cd flaskenv1/Scripts 然后执行activate

 b.Linux/Mac下激活：

 cd flaskenv1/Scripts 执行source activate

  • 第四步：退出环境

deactivate

二、flask安装与使用

1.安装：激活环境

2.执行如下命令：

pip install flask

三、pip使用

  • 查询安装的库：

pip list/ pip freeze

  • 安装内容：

pip install xxx

  • 安装文件：

pip install -r requirement.txt

四、启动

  • flask自带启动方式

app.run(host, port, debug)

  • 使用flask-script库启动

 1.先安装flask-script库

 pip install flask-script

 2.调用Manager

  • 启动命令：

python xxx.py runserver -p -h -d

五、路由规则

1.定义格式：<转换器（类型）：参数名>

2.转换器类型：

  int：指定整型

  string：指定字符串，也是默认的类型

  float：指定浮点类型

注意：string:name其实和是一样的

六、请求与响应

1.请求
  • 获取参数：

 ①get请求获取参数：

 request.args

 request.args[key]

 request.args.get(key)

 ②post请求获取参数：

 request.form

 request.form[key]

 request.form.get(key)

  • 获取请求路径：

request.path

  • 获取cookies：

request.cookies

  • 获取上传文件：

request.files

reuqest.files[key]

request.files.get(key)
